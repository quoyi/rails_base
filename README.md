# RailsBase

提供用户注册认证、角色管理、组织管理等基础功能。

## Usage

* Copy over all the migrations from the engine

  ```bash
  rails railties:install:migrations
  ```

* Execute migrations

  ```bash
  rails db:migrate
  ```

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rails_base'
```

And then execute:

```bash
bundle
```
